## 基于 BERT 的抽取式问答系统

### 项目简介

这是一个基于 **BERT-base-chinese** 预训练模型搭建的抽取式问答（QA）系统，针对中文问答任务（以 CMRC2018 数据集为例）进行训练和预测。系统能够从给定的上下文文本中，提取出对应问题的答案，支持超长文本的滑动窗口处理，避免答案因文本截断丢失。

### 数据集说明

**CMRC2018 数据集，这是中文领域经典的**抽取式机器阅读理解数据集**，专门用于训练和评估问答模型的性能。

数据示例：json 格式

### 核心功能：

- 加载 CMRC2018 格式的 JSON 问答数据

- 数据预处理（分词、长文本拆分、答案位置对齐）

- BERT 模型训练与最优模型保存

- 模型效果评估（验证集损失计算）

- 长文本支持的问答预测

### 测试结果

选取验证集中的三个问题进行测试：

```
问题 1: 《战国无双3》是由哪两个公司合作开发的？
预期答案: 光荣和ω-force
预测答案: 光荣和ω-force
匹配: True

问题 2: 男女主角亦有专属声优这一模式是由谁改编的？
预期答案: 村雨城
预测答案: 任天堂游戏谜之村雨城
匹配: True

问题 3: 战国史模式主打哪两个模式？
预期答案: 「战史演武」&「争霸演武」
预测答案: 主打两大模式「战史演武」&「争霸演武」。
匹配: True
```
