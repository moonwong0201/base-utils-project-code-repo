基于 Transformer 的文本意图分类模型

## 项目功能 

1. 基于 jieba 实现中文文本分词与词汇表构建； 
2. 实现 Transformer 编码器的文本特征提取；
3. 在训练过程中自动保存测试集最优准确率模型（将整个数据集分成 80% 训练集，20% 测试集）； 
4. 提供了单条文本预测接口，支持批量测试用例验证。
