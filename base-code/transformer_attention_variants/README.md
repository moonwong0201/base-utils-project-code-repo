### 三种注意力类型

| 注意力类型 |            Q/K/V Head 数            |          核心目标           |
| :--------: | :---------------------------------: | :-------------------------: |
|    MHA     |       Q/K/V Head 数量完全相同       |    精度优先（效果最好）     |
|    MQA     |    Q Head 多，K/V Head 固定为 1     | 速度 / 显存优先（开销最小） |
|    GQA     | Q Head 分组，每组共享 1 组 K/V Head |       平衡精度和速度        |
